⚙️ Configuración inicial (UNA SOLA VEZ)

  1. Crear cuenta gratuita en https://www.emailjs.com (200 correos/mes gratis)
  2. Conectar tu correo (Gmail/Outlook) → obtienes un Service ID
  3. Crear una plantilla de correo → obtienes un Template ID
  4. Copiar tu Public Key desde "Account → API Keys"
  5. Reemplazar las constantes EMAILJS_* en el script de abajo
  6. ¡Listo! Los correos se envían directamente desde el navegador, sin backend
