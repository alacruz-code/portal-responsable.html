# 🛡️ Sistema de Auditoría Interna — Mayoreo 2026

Sistema integrado de gestión de hallazgos de auditoría con dashboard ejecutivo, 
portal de respuesta para responsables y notificaciones automáticas.

## 🌐 Accesos directos

- 📊 **Dashboard Ejecutivo**: [[Ver dashboard](./index.html)](https://alacruz-code.github.io/portal-responsable.html/)
- 👤 **Portal Responsables**: [[Acceder](./portal-responsable.html)](https://alacruz-code.github.io/portal-responsable.html/portal-responsable.html)
- 📧 **Centro de Envíos**: [[Gestionar correos](./integracion-emails.html)](https://alacruz-code.github.io/portal-responsable.html/integracion-emails.html)

## 🚀 Características

- ✅ Dashboard con filtros dinámicos y heat map de riesgos
- ✅ Seguimiento en tiempo real de hallazgos
- ✅ Sistema de alertas automáticas por vencimiento
- ✅ Portal independiente con tokens únicos por responsable
- ✅ Envío automatizado de correos vía EmailJS
- ✅ Escalamiento automático a Gerencia

## 📊 Empresas integradas

🇻🇪 Febeca · Beval · Sillaca · Prisma  
🇨🇴 Mundial de Partes  
🇨🇷 Cofersa  

## 🛠️ Tecnología

HTML5 · CSS3 · JavaScript · Chart.js · EmailJS
